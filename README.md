# Algorithm_Python

Python으로 백준 알고리즘 문제를 풀이하는 연습 저장소입니다. 현재는 백트래킹 유형 문제를 중심으로 풀이 파일이 정리되어 있습니다.

## 문제 범위

- 백준 `15649`, `15650`, `15651`, `15652`: N과 M 계열 백트래킹 문제
- 백준 `9663`: N-Queen 문제
- `Test.py`: 풀이 실험 또는 확인용 파일

## 폴더 구조

```text
baekjoon/
└── 백트래킹/
    ├── BOJ15649.py
    ├── BOJ15650.py
    ├── BOJ15651.py
    ├── BOJ15652.py
    └── BOJ9663.py
```

## 실행 방법

별도 의존성 파일은 없으며, Python 3 환경에서 원하는 풀이 파일을 직접 실행합니다.

```bash
python3 baekjoon/백트래킹/BOJ15649.py
```

대부분의 온라인 저지 풀이처럼 표준 입력을 받아 동작합니다.

## 메모

이 저장소는 알고리즘 개념 복습과 코딩 테스트 대비를 위한 코드 모음입니다. 아직 플랫폼별 전체 분류나 자동 테스트 설정은 포함되어 있지 않습니다.
